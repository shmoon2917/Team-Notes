# 문자열 재정렬

- 알파벳 대문자와 숫자(0 ~ 9)로만 구성된 문자열이 입력으로 주어집니다. 이때 모든 알파벳을 오름차순으로 정렬하여 이어서 출력한 뒤에, 그 뒤에 모든 숫자를 더한 값을 이어서 출력합니다.

- 예를 들어 K1KA5CB7 이라는 값이 들어오면 ABCKK13을 출력합니다.

- _입력 조건)_

  - 첫째 줄에 하나의 문자열 S가 주어집니다.

- _출력 조건)_
  - 첫째 줄에 문제에서 요구하는 정답을 출력합니다.

<br></br>

## 풀이

```python
answer = ''
s = sorted(input())
sum = 0
for cha in s:
  if cha.isalpha():
    answer += cha
  else:
    sum += int(cha)

answer += str(sum)
print(answer)
```
